Personal Projects

One* Night at Miku's

Platforms: Windows | Linux
Project Length: Current
Engine and Tools: Monogame

One* Night at Miku's is an upcoming action focused FNaF fangame that combines classic sit-and-survive gameplay with Rougelike elements.

Unnamed Project "Sokoban"

Platforms: Windows
Project Length: ~2 months
Engine and Tools: Monogame | Sesame

“Sokoban” was a narrative puzzle adventure game. Players would solve Sokoban-style block puzzles in order to traverse a mysterious world governed by game logic and forces just beyond the screen.

Undertale Battle Engine

Platforms: Windows
Project Length: ~1 month
Engine and Tools: Java Swing

Undertale Battle Engine is a game engine used to play custom-made Undertale battles; a mix between JRPG turn-based combat and casual danmaku gameplay.

Sha's Tetris

Platforms: Windows
Project Length: ~2 month
Engine and Tools: Java Swing

Sha's Tetris is a simple Tetris clone based on Tetris (1989) for the Game Boy.

SSRG

Platforms: Windows
Project Length: ~1 month
Engine and Tools: Java Swing

SSRG is a four-lane, tile rhythm game in the style of Piano Tiles or Guitar Hero in which players match button presses to the notes of a song.

Collaborative Projects/Game Jam Games

Unnamed Project "Project One"

Platforms: Windows
Team Size: 2
Project Length: ~3 months
Responsibilities: Project Lead | Game Designer | Programmer | Writer

“Project One” was a visual-novel narrative experience centered around serving ice cream to an eccentric cast of characters. Players would form connections with customers over the course of multiple work days and slowly piece together the true nature of the world they find themselves in.

Adrenaraid

Platforms: Windows
Team Size: 3
Project Length: ~1 week
Responsibilities: Project Lead | Game Designer | Programmer

Adrenaraid is a fast-paced arcade action game in which an adrenaline junky rat weaves between supermarket shoppers as it steals food. Made as part of Trijam #270.

Torch

Platforms: Windows
Team Size: 2
Project Length: ~3 days
Responsibilities: Lead Game Designer | Assistant Programmer | Graphic Designer

Torch is a 2D survival sandbox game akin to Terraria. However, you play as the world itself, attempting to thwart the progression of an AI-controlled player-character. Made as part of GMTK Game Jam 2023.
